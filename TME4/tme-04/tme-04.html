    """ desc_set * label_set * array * function * int * int * str -> NoneType
        Note: le classifieur linaire est donn sous la forme d'un vecteur de poids pour plus de flexibilit
    """
    # ETAPE 1: construction d'une grille de points sur tout l'espace dfini par les points du jeu de donnes
    mmax=desc_set.max(0)
    mmin=desc_set.min(0)
    x1grid,x2grid=np.meshgrid(np.linspace(mmin[0],mmax[0],step),np.linspace(mmin[1],mmax[1],step))
    grid=np.hstack((x1grid.reshape(x1grid.size,1),x2grid.reshape(x2grid.size,1)))
    #
    # Si vous avez du mal  saisir le concept de la grille, dcommentez ci-dessous

    # if True:
    #    return

    # ETAPE 2: calcul de la prediction pour chaque point de la grille
    res=np.array([kernel(grid[i,:])@w for i in range(len(grid)) ])
    # pour les affichages avancs, chaque dimension est prsente sous la forme d'une matrice
    res=res.reshape(x1grid.shape) 
    #
    # ETAPE 3: le trac
    #
    # CHOIX A TESTER en dcommentant:
    # 1. lignes de contours + niveaux
    if forme <= 2 :
        fig, ax = plt.subplots() # pour 1 et 2
        ax.set_xlabel('X_1')
        ax.set_ylabel('X_2')

        CS = ax.contour(x1grid,x2grid,res)
        ax.clabel(CS, inline=1, fontsize=10)
    #

        CS = ax.contour(x1grid,x2grid,res, levels=[0], colors='k')

        surf = ax.plot_surface(x1grid,x2grid,res, rcount=rcount, ccount=ccount, facecolors=colors, shade=False)
        surf.set_facecolor((0,0,0,0))
    #

def get_contour_data(X, Y, Z, levels=None):
    if levels == None:
        cs = plt.contour(X, Y, Z) # matplotlib => bokeh
    else:
        cs = plt.contour(X, Y, Z, levels) # matplotlib => bokeh
    xs = []
    ys = []
    xt = []
    yt = []
    col = []
    text = []
    isolevelid = 0
    for isolevel in cs.collections:
        isocol = isolevel.get_color()[0]
        thecol = 3 * [None]
        theiso = str(cs.get_array()[isolevelid])
        isolevelid += 1
        for i in range(3):
            thecol[i] = int(255 * isocol[i])
        thecol = '#%02x%02x%02x' % (thecol[0], thecol[1], thecol[2])

        for path in isolevel.get_paths():
            v = path.vertices
            x = v[:, 0]
            y = v[:, 1]
            xs.append(x.tolist())
            ys.append(y.tolist())
            xt.append(x[len(x) // 2])
            yt.append(y[len(y) // 2])
            text.append(theiso)
            col.append(thecol)

    data = dict({'xs': xs, 'ys': ys, 'line_color': col,'xt':xt,'yt':yt,'text':text})
    return data

ADALINE: la minimisation des moindres carrés¶

La fonction la plus connue est la fonction des moindres carrs: $$ \mathcal C = \sum_{i=1}^N (f(\mathbf x_i) - y_i)^2$$ Il faut que $f(\mathbf x_i)$ soit le plus proche possible de $y_i$ pour faire baisser ce cot.

Dans le cas o $f$ est une fonction linaire, on peut rsoudre le problme de minimisation de la fonction de cot par descente de gradient stochastique, un peu comme pour le perceptron:

Algorithme ADALINE:

  1. Initialisation de $\mathbf w$ alatoire
  2. Boucle
    1. Tirage d'un point i
    2. Calcul du gradient $\nabla_{\mathbf w} \mathcal C$ (cf ci-dessous)
    3. $\mathbf w \leftarrow \mathbf w - \epsilon \nabla_{\mathbf w} \mathcal C $
    4. Test de convergence + sortie

Avec: $$\nabla_{\mathbf w} \mathcal C = \mathbf x_i^T (\mathbf x_i \mathbf w - y_i) \in \mathbb R^d $$

  1. Coder l'algorithme ADALINE et vrifier qu'il fonctionne bien sur des donnes jouet. Pour vous aider, nous vous donnons le squelette de la classe crer dans la bote suivante. Veuillez noter l'utilisation de classif.Classifier pour dfinir la classe mre de ce nouveau classifier.

  2. Une fois que le code est fonctionnel, rajouter cette nouvelle classe dans Classifiers.py. Attention: dans le fichier Classifiers.py, il ne faudra plus utiliser classif. devant le nom de Classifier mais crire:

     class ClassifierADALINE(Classifier):
 ...

  3. Vrifier que vous tes capable de tracer l'volution des poids du classifeur comme pour le perceptron.

ADALINE analytique¶

Contrairement au perceptron, il est possible de calculer une solution analytique pour ADALINE. En effet, on peut montrer que l'annulation du gradient correspond au problme suivant:

$$ \underbrace{X^T X}_{\in \mathbb R^{d \times d}} \ \underbrace{\mathbf w}_{\in \mathbb R^{d}} = \underbrace{X^T Y}_{\in \mathbb R^{d}}$$

La fonction np.linalg.solve permet justement de rsoudre un tel systme d'quations linaires.

  1. Utiliser la documentation pour appliquer np.linalg.solve sur le problme qui nous intresse et vrifier que la solution est bien intressante.
  2. Ecrire la classe ClassifierADALINE2 qui implmente le classifieur ADALINE utilisant cette approche pour calculer la solution analytique.
  3. Une fois fonctionnel, intgrer ce code dans le fichier Classifiers.py

Note: cet algorithme n'est plus itratif, il n'y a plus de learning_rate ni d'itration max ou d'historique...

Validation Croisée¶

La procdure de la validation croise est au centre de la plupart des applications de machine learning: il est temps pour nous de travailler sur une implmentation.

L'ide est de concevoir la fonction suivante:

Xapp,Yapp,Xtest,Ytest = crossval(X, Y, n_iterations, iteration)
  • X, Y sont les donnnes mlanges.
  • n_iterations le nombre d'ensembles de donnes de test disjoints total.
  • iteration l'itration concerne: on ne renvoit pas les mmes donnes en fonction des itrations.

Aprs sparation des indices en n_iterations groupes, isoler 1 groupe pour le test et les autres pour l'apprentissage.

Note Pour mlanger les donnes, vous pourrez utiliser les commandes suivantes:

index = np.random.permutation(len(X)) # mlange des index
Xm = X[index]
Ym = Y[index]

Retour sur le multi-classes¶

On ne va pas dfinir une version multi-classe par classifieur binaire... Il est possible de dfinir un classifieur gnrique multi-classe partir de classifieur binaire (design pattern composite) en respectant le schma suivant:

  1. ClassifierMultiOAA prend un classifieur binaire en argument lors de la cration. Il initialise une liste vide de classifieurs.

  2. Dans la mthode train, penser cloner le classifeur de rfrence nCl fois. Il faut utiliser la mthode deepcopy du module standard python copy

  3. Dans un boucle for, redfinir les tiquette $ytmp$ et apprendre des classifieurs binaires

  4. Dans score faire appel aux mthodes score des classifieurs binaires et stocker les rsultats

  5. Dans predict renvoyer l'argmax des scores.
    Note: en faisant cette opration, on fait l'hypothse que les tiquettes sont dfinies sur $[0,nCl]$. Pour lever cette hypothse, il faudrait stocker les classes et convertir le rsultat de argmax.

  6. La fonction accuracy de la classe mre doit tre compatible avec les dfinitions prcdentes... Par exemple:

    def accuracy(self, desc_set, label_set):
     yhat = np.array([self.predict(x) for x in desc_set])
     return np.where(label_set == yhat, 1., 0.).mean()
